Automated TixChange Version: 20.6.0
Release Date: Jun 20th, 2020
Key features in this release:
  Open Tracing - Integrated with Jaeger Backend and Jaeger Client
     - New UMA that stitches UMA instrumented java traces with Open Tracking Spans 
     - A new service is added to TixChange "pricingService" that calls a new NodeJS backend that is instrumented with Jaeger client
  Jenkins - more improvements and bug fixes and include automated trigger with alternate pass and fail build.
